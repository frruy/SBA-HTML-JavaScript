Our Story

In Spring Boot, a composed annotation is an annotation created by combining multiple existing annotations. This allows you to create a single custom annotation that encapsulates the behavior of multiple annotations, making your code cleaner and more expressive.

Annotation Description
@TransactionalComponent A composed annotation combining the behavior of @Component and @Transactional. It can be used wherever you would normally use @Component or @Transactional.
@SecureLogging A composed annotation combining the behavior of @Component and adding secure logging functionality. It ensures that sensitive log messages are encrypted before being logged.
@CachingService A composed annotation combining the behavior of @Service and @Cacheable. It marks a service as cacheable, allowing method calls to be cached for improved performance.
